🐘 PHP Logic

The calculator processes arithmetic via $_POST form submission. Each calculation is stored in MySQL using mysqli_query(). Division by zero is caught with a PHP conditional before executing.

🗄️ MySQL Schema

Table calc_history stores every calculation: id INT AUTO_INCREMENT, num_a FLOAT, op VARCHAR(1), num_b FLOAT, result FLOAT, created_at TIMESTAMP.
